Representación

String Map se representa con tupla<head: puntero(Nodo), cantClaves: nat>

donde Nodo es tupla <clave: string, padre: puntero(Nodo), hijos: arreglo(puntero(Nodo)), cantHijos: nat, par: tupla(string, T)>>

rep: string_map sm bool
rep(sm)

  • sm.cantClaves == #clavesSM(sm.head)
  • cantHijosOk(sm.head)
  • consistenciaPadreHijo(sm.head, sm.head.hijos)
  • caminoValidoClaveDePar(sm.head, sm.head, sm.head.hijos)

abs: string_map sm diccionario(string, T)
abs(sm) d : diccionario |

  • claves(d) == clavesSM(sm.head)
  • ( cl claves(d)) (definido(cl, sm) obtener(cl, d) == significado(cl, sm))

cantHijosOk : Nodo n bool

cantHijosOk(n) if n != nil then n.cantHijos == contarHijos(n.hijos) hijosDeHijos(n.hijos) else True fi

hijosDeHijos : arreglo(nodo) an bool

hijosDeHijos(an) if vacia?(an) then True else cantHijosOk(prim(an)) hijosDeHijos(fin(an)) fi

contarHijos : arreglo(nodo) an nat

contarHijos(an) if vacia?(an) then 0 else if prim(an) != nil then 1 + contarHijos(fin(an)) else 0 + contarHijos(fin(an)) fi fi

consistenciaPadreHijo : Nodo n x arreglo(nodo) an bool

consistenciaPadreHijo if vacia?(an) then True else if prim(an) != nil then n == prim(an).padre n.hijos[ord(prim(an).clave))] == prim(an) consistenciaPadreHijo(n, fin(an)) consistenciaPadreHijo(prim(an), prim(an).hijos) else consistenciaPadreHijo(n, fin(an)) fi fi

caminoValidoClaveDePar : Nodo r x Nodo n x arreglo(nodo) an bool

caminoValidoClaveDePar if vacia?(an) then True else if prim(an) != nil then if prim(an).par != nil then buscar(π1(prim(an)), π1(prim(an)) r) caminoValidoClaveDePar(r, n, fin(an)) caminoValidoClaveDePar(r, prim(an), prim(an).hijos) else caminoValidoClaveDePar(r, n, fin(an)) caminoValidoClaveDePar(r, prim(an), prim(an).hijos) fi else caminoValidoClaveDePar(r, n, fin(an)) fi

clavesSM if n != NIL then if n.par != NIL then Ag(π1(n.par), ø) U recorrerHijos(n.hijos) else recorrerHijos(n.hijos) fi else ø fi

template<typename T>
class string_map< T >

Implementacion de map<string,T> sobre Trie.

Asume de T:

  • tiene constructor por copia
  • tiene operador ==
  • solo permite utilizar el operator[] si T tiene constructor por defecto

se explica con TAD Diccionario(string, T)
